Quantum Random Walk on a Line: Ballistic Spreading

A comparison of the classical and quantum walk dynamics, focusing on the significantly enhanced spread of the quantum mechanical analogue.

The **Quantum Random Walk (QRW)** is the quantum mechanical counterpart to the classical random walk. The key difference lies in the replacement of classical probability distributions with **quantum superposition** and **interference**. The walk's state is defined on a Hilbert space composed of position and a 'coin' degree of freedom.

Spread Rate: Quantum vs. Classical

After $t$ steps, the standard deviation $\sigma$ (the spatial spread) of the distribution follows distinct laws:

  • **Classical Random Walk (CRW):** Diffusive spread, $\sigma \propto \sqrt{t}$.
  • **Quantum Random Walk (QRW):** Ballistic spread, $\sigma \propto t$.
This quadratic speedup is what makes QRWs a powerful tool for developing faster quantum search algorithms.

1. The Two-Part Operator

Each step of the discrete-time QRW on a line is governed by a unitary operator, $\hat{U}$, which is a sequential application of a Coin Operator ($\hat{C}$) and a Shift Operator ($\hat{S}$).

The Full Quantum Walk Operator $\hat{U}$

$$\hat{U} = \hat{S} \cdot (\hat{C} \otimes \hat{I})$$

where $\hat{I}$ is the identity operator acting on the position space, ensuring the coin flip happens independently of location.

2. The Shift Operator $\hat{S}$ and Coin Operator $\hat{C}$

The **Shift Operator** moves the walker one step left or right, depending on the coin state (up $|\uparrow\rangle$ or down $|\downarrow\rangle$). The most common **Coin Operator** is the Hadamard gate, $\hat{H}$.

Hadamard Coin $\hat{C}$ and Conditional Shift $\hat{S}$

$$\hat{C} = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}$$

The conditional shift operator $\hat{S}$ acts on the position basis $|x\rangle$ as:

$$\hat{S} (|\uparrow\rangle \otimes |x\rangle) = |\uparrow\rangle \otimes |x+1\rangle \\ \hat{S} (|\downarrow\rangle \otimes |x\rangle) = |\downarrow\rangle \otimes |x-1\rangle$$

3. Iteration and Probability Calculation

The evolution is performed by repeated application of the unitary operator: $|\Psi(t)\rangle = \hat{U}^t |\Psi(0)\rangle$. The final distribution is found by summing the probabilities of the two coin states at each position $x$.

Python Implementation: QRW Iteration

# qrw_simulation.py: Quantum Random Walk Iteration
import numpy as np

# --- System Parameters ---
T_STEPS = 50        # Number of walk steps
N_POSITIONS = 2 * T_STEPS + 1 # Total positions to track (e.g., -50 to 50)
CENTER = T_STEPS
PSI = np.zeros((N_POSITIONS, 2), dtype=complex) # State vector: [position, coin state]

# Initial state: Position 0, Equal superposition coin state (|0> + |1>)/sqrt(2)
PSI[CENTER, 0] = 1 / np.sqrt(2) # Coin Up (a_x)
PSI[CENTER, 1] = 1 / np.sqrt(2) # Coin Down (b_x)

# Hadamard Coin Operator
H = (1 / np.sqrt(2)) * np.array([[1, 1], [1, -1]])

def quantum_walk_step(psi_t):
    """Performs one step of the Quantum Random Walk."""
    psi_next = np.zeros_like(psi_t)
    N = psi_t.shape[0]

    # 1. Apply Coin Operator (Local operation at each position x)
    psi_coin = np.dot(psi_t, H)

    # 2. Apply Conditional Shift Operator (Moves state to new positions)
    for x in range(N):
        # a_x (coin up) moves right (x -> x+1)
        if x + 1 < N:
            psi_next[x + 1, 0] += psi_coin[x, 0]
        # b_x (coin down) moves left (x -> x-1)
        if x - 1 >= 0:
            psi_next[x - 1, 1] += psi_coin[x, 1]
            
    return psi_next

# Run the simulation for T_STEPS
current_psi = PSI
for t in range(T_STEPS):
    current_psi = quantum_walk_step(current_psi)

# Probability distribution P(x) = |a_x|^2 + |b_x|^2
probability_dist = np.sum(np.abs(current_psi)**2, axis=1)

print(f"QRW simulation complete after {T_STEPS} steps.")
print(f"Max probability position: {np.argmax(probability_dist) - CENTER}")
