3D Double-Well Potential: Tunneling and Stationary States

A technical deep dive into the stationary states and probability densities of a single particle confined by two identical potential wells, a critical model for understanding quantum tunneling and coherence.

The double-well potential is the simplest model that exhibits the quantum phenomenon of **tunneling** and is analogous to the double-slit experiment in momentum space. When the barrier separating the two Gaussian wells is finite, the particle's wave function is not confined to a single well. Instead, the ground state and first excited state become a pair of states: a symmetric state ($n=0$) and an anti-symmetric state ($n=1$), separated by a small **tunneling energy splitting** ($\Delta E$).

1. The Physics: The Time-Independent Schrödinger Equation

The stationary states $\Psi_n(x,y,z)$ and their energies $E_n$ are found by solving the time-independent Schrödinger equation: $\hat{H}\Psi_n = E_n\Psi_n$. The Hamiltonian consists of the kinetic energy ($\hat{T}$) and the double Gaussian potential ($\hat{V}$).

The Double-Well Hamiltonian and Potential

$$\hat{H} = \hat{T} + \hat{V}(x,y,z)$$ $$V(x,y,z) = -V_0 \left(e^{-\frac{r_A^2}{2\sigma^2}} + e^{-\frac{r_B^2}{2\sigma^2}}\right)$$

Where $r_A$ and $r_B$ are the distances from the coordinate $(x,y,z)$ to the center of well A and well B, respectively. $V_0$ is the depth, and $\sigma$ is the width of the wells.

2. C++ Kernel Concept: Potential and State Calculation

In a high-performance computing environment, the potential and the components of the wave function ($\Psi_n$) would be calculated in parallel across a 3D grid, or lattice, using a CUDA kernel or similar parallel framework.

Kernel for Potential and Probability Density

// kernel_wavefunction.cu: CUDA Kernel for 3D Stationary State (n=0 approximation)
__device__ double gaussian_well(double x, double y, double z, double center_x, double V0, double sigma) {
    // Calculates the local potential contribution from one well
    double r_sq = pow(x - center_x, 2) + pow(y, 2) + pow(z, 2);
    return -V0 * exp(-r_sq / (2.0 * pow(sigma, 2)));
}

__global__ void density_kernel(const double* grid_coords, double* density_output, int L, double V0, double sigma, double D) {
    // L is the total number of grid points (Nx*Ny*Nz)
    int idx = blockIdx.x * blockDim.x + threadIdx.x;
    if (idx < L) {
        // Map 1D index to 3D coordinates (x, y, z)
        double x = grid_coords[idx * 3 + 0];
        double y = grid_coords[idx * 3 + 1];
        double z = grid_coords[idx * 3 + 2];

        // phi_A and phi_B are localized component wave functions (Gaussian approximation)
        double phi_A = exp(-((pow(x + D, 2) + pow(y, 2) + pow(z, 2)) / (2.0 * pow(sigma, 2))));
        double phi_B = exp(-((pow(x - D, 2) + pow(y, 2) + pow(z, 2)) / (2.0 * pow(sigma, 2))));

        // Ground State (n=0): Symmetric superposition Psi_0 = phi_A + phi_B
        double psi_0 = phi_A + phi_B;
        
        // Probability Density |Psi|^2
        density_output[idx] = pow(psi_0, 2);
    }
}
// End of C++ Kernel Concept.

3. Python Analysis: Energy Splitting and Eigenvalues

For a simpler, one-dimensional version of the double-well, numerical methods in Python, like the Finite Difference Method (FDM) and linear algebra libraries, are used to solve the single-particle system and find the energy eigenvalues $E_n$. The magnitude of the small energy gap ($\Delta E$) between $E_0$ and $E_1$ quantifies the tunneling rate.

Python Numerical Analysis for 1D Tunneling

# python_eigen_analysis.py: 1D Double-Well Energy Calculation
import numpy as np
import scipy.linalg

# --- System Parameters (1D Analog) ---
N = 500             # Number of grid points
X_max = 10.0        # System extent: [-X_max, X_max]
V0 = 5.0            # Potential depth
D = 2.0             # Well position offset ( +/- D)
sigma = 0.5         # Well width

# 1. Setup Grid and Step Size
x = np.linspace(-X_max, X_max, N)
dx = x[1] - x[0]
dx2 = dx**2
hbar = 1.0          # Set hbar=1, mass=1 for simplicity
m = 1.0

# 2. Define the Potential V(x)
def V_double_well(x, V0, D, sigma):
    # Two Gaussian wells centered at -D and +D
    return -V0 * (np.exp(-(x + D)**2 / (2 * sigma**2)) + np.exp(-(x - D)**2 / (2 * sigma**2)))

V = V_double_well(x, V0, D, sigma)

# 3. Construct the Hamiltonian Matrix H (using Finite Difference Method for kinetic term)
# H = T + V
# The kinetic term T is proportional to the second derivative operator (tridiagonal)
# T_ii = hbar^2 / (m * dx^2)
# T_i,i+1 = T_i+1,i = -hbar^2 / (2 * m * dx^2)
T_diag = hbar**2 / (m * dx2)
T_off_diag = -hbar**2 / (2 * m * dx2)

# Build the kinetic matrix T
T = (T_diag * np.eye(N)) + (T_off_diag * (np.diag(np.ones(N - 1), 1) + np.diag(np.ones(N - 1), -1)))

# Add the potential V to the diagonal
H = T + np.diag(V)

# 4. Solve the Eigenvalue Problem
# We find the eigenvalues (energies E_n) and eigenvectors (wave functions Psi_n)
eigenvalues, eigenvectors = scipy.linalg.eigh(H)

# 5. Analyze Results for Tunneling Splitting
E0 = eigenvalues[0] # Ground state energy
E1 = eigenvalues[1] # First excited state energy
tunneling_splitting = E1 - E0

print("-" * 35)
print(f"--- 1D Double-Well Analysis ---")
print(f"Ground State Energy (E0): {E0: .4f}")
print(f"First Excited Energy (E1): {E1: .4f}")
print(f"Tunneling Energy Splitting (Delta E): {tunneling_splitting: .4e}")
print("-" * 35)
# This small, non-zero splitting confirms quantum tunneling between the wells.
# The eigenvectors[:, 0] and eigenvectors[:, 1] represent the Psi_0 and Psi_1 states.
# End of Python Numerical Analysis.
