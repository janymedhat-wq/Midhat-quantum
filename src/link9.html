Bloch Sphere Visualization for Single Qubits

A guide to representing and manipulating the state of a single qubit in 3D space, demonstrating the action of common quantum gates.

The **Bloch Sphere** provides a geometric representation of the pure state space of a single qubit. The north pole represents the computational basis state $|0\rangle$, and the south pole represents $|1\rangle$. Any point on the surface of the unit sphere corresponds to a valid quantum state $|\Psi\rangle$.

Qubit State Vector $|\Psi\rangle$

A single qubit state is a superposition of $|0\rangle$ and $|1\rangle$, parametrized by two angles, $\theta$ (polar) and $\phi$ (azimuthal):

$$|\Psi\rangle = \cos\left(\frac{\theta}{2}\right)|0\rangle + e^{i\phi}\sin\left(\frac{\theta}{2}\right)|1\rangle$$

1. Pauli Gates: Rotations

The Pauli gates ($\hat{X}, \hat{Y}, \hat{Z}$) are the fundamental single-qubit operations. Geometrically, they correspond to $\pi$ (180-degree) rotations about the respective axes of the Bloch sphere.

Pauli Gate Matrices

$$\hat{X} = \begin{pmatrix} 0 & 1 \\ 1 & 0 \end{pmatrix} \quad \hat{Y} = \begin{pmatrix} 0 & -i \\ i & 0 \end{pmatrix} \quad \hat{Z} = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$$

2. The Hadamard Gate

The **Hadamard gate ($\hat{H}$)** is crucial for creating superposition. It maps the computational basis states ($|0\rangle, |1\rangle$) to the superposition states ($|+\rangle, |-\rangle$), which lie on the equator of the Bloch sphere (the $X$ axis).

Hadamard Gate Matrix

$$\hat{H} = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 & 1 \\ 1 & -1 \end{pmatrix}$$

3. Matrix Multiplication Implementation

In a simulation environment, applying a quantum gate is achieved by multiplying the $2 \times 2$ unitary gate matrix by the $2 \times 1$ state vector. This operation is trivially parallelizable across multiple qubits in a larger simulation.

Conceptual JavaScript for Gate Application

// qiskit_core.js: Applying a Gate via Matrix Multiplication
/**
 * Applies a 2x2 gate matrix to a 2-element qubit state vector.
 * @param {Array>} gate - The 2x2 unitary matrix (e.g., Pauli X).
 * @param {Array} state - The [alpha, beta] state vector.
 * @returns {Array} The new state vector.
 */
function applyGate(gate, state) {
    let newState = [new Complex(0, 0), new Complex(0, 0)];

    // New alpha = G[0][0]*alpha + G[0][1]*beta
    newState[0] = gate[0][0].multiply(state[0]).add(gate[0][1].multiply(state[1]));
    
    // New beta = G[1][0]*alpha + G[1][1]*beta
    newState[1] = gate[1][0].multiply(state[0]).add(gate[1][1].multiply(state[1]));
    
    return newState;
}
